Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b > (não sêmantica)

Nessa frase temos um termo em destaque usando a tag <strong > (semântica)

Ítalico / Ênfase

Nessa frase temos um termo em ítalico usando a tag < i > (não semântica)

Nessa frase temos um termo em ênfase usando a tag < em > (semântica)

Texto marcado

Podemos criar támbem um texto marcado usando a tag < mark >

Texto grande e pequeno

Estamos criando um texto grande com a tag < big > (obsoleta) e um texto pequeno com a tag < small > neste parágrafo

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado com a tag < del >

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag < ins >

Existe támbem o sublinhado com a tag < u > (não sêmantica)

Texto sobrescrito

Para inserir coisas do tipo: x20+3 com a tag < sup >

Texto subscrito

Para inserir coisas do tipo: H2O com a tag < sub >
